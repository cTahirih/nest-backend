## Installation

```bash
$ npm install
```
## Running the Backend

```bash
$ docker compose up -d
```

Copy .env.template and rename to .env


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
